Current Projects

Stock dashboard

The Stock Dashboard is a portfolio management tool designed to optimize how users track and organize their investments. It consolidates stocks from multiple brokers into a single interface, allowing users to input and monitor all their holdings in one place. Built on an PostgresSQL database, it securely manages multiple users and stock records while ensuring reliable storage and retrieval. The system provides live stock details upon refresh, enabling users to stay up to date with price changes, analyze performance, and view their total holdings in real time. By centralizing information, the dashboard reduces the need to manually track assets across different platforms, improving both accuracy and efficiency. Looking ahead, the platform is designed to evolve into a decision-support tool, with planned features such as personalized stock suggestions and portfolio optimization insights.

Excel Budget Planner

The Budget Planner is an interactive tool designed to help users manage and organize important dates such as birthdays with ease. It offers a command-based interface that allows you to add new entries, view the complete list, delete specific records, and search for names quickly. Users can also save their birthday list to a file for long-term storage or reload it later for continued use. To enhance usability, the system includes an “echo” mode that confirms actions as they are performed, as well as a “help” command that guides users through all available features. By combining simplicity with practical functionality, the Budget Planner ensures that important dates are never forgotten and can be accessed in a structured, organized manner.

Ultrasonic Object Detection Robot

The Ultrasonic detection robot The Ultrasonic Detection Robot is an upcoming Arduino-based pathfinder that uses ultrasonic sensors to detect obstacles and choose safe routes. Designed as a foundation for future robotics work, it demonstrates autonomous navigation with plans to expand into mapping and smarter decision-making.
